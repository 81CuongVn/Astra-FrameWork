# Astra-FrameWork
🚀 A pen-testing framework that's out of this world

## Setup
```bash
git clone https://github.com/NotReeceHarris/Astra-FrameWork.git
```
```bash
cd Astra-FrameWork
```
```bash
sudo python3 astra.py --setup
```
